### テーマを選んだ理由
<!-- 【補足説明】 -->
<!-- - ですます調で記載しましょう。READMEファイルは企業様も見られます。 -->
<!-- - ３文以上記載しましょう。 -->
旅行好きでよくサイトなどインスタで旅先の情報を調べることがありました。

ですが実際に旅行計画を立てるのですが計画通りに行かないことが複数回ありました。

そんな時に他の人がどんな計画でどのような時間の使い方をしていたのか参考になるものが欲しかったり
どのようなスポットがありどんな場所でどの世代にニーズがあるかなどの情報が欲しかったのでこのサイトを
制作しようと思いました。実際に友人も困っていたのであったら良いなと思いこのテーマにしました。
